# Smart Ads System

AI-powered demographic-targeted advertising system with real-time face detection.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- TensorFlow.js
- face-api.js

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Deployment

Build the project for production:

```sh
npm run build
```
